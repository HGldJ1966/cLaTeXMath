Detailed Description

Represent a 2D graphics context, all the TeX drawing operation would be on it. Current supported affine transformation is scale, translation and rotation, but note that the scaling on y direction will be selected as base if the scaling on x and y direction are different when drawing a character, so you should never use a different scaling unless you are really sure that the coordinate is positioned correctly (i.e. draw a hyphen).
